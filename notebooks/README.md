# Jupyter Notebooks for Silence Pattern Decoder

本目录包含用于学习和实验的交互式 Jupyter Notebook。

## 笔记本列表

| 笔记本 | 描述 |
|--------|------|
| `01-introduction.ipynb` | 项目介绍、基本概念和环境设置 |
| `02-consensus-detection.ipynb` | 公识检测实验，展示如何检测群体中的隐式公识模式 |
| `03-influence-analysis.ipynb` | 少数派影响和压迫分析，探索少数群体对多数群体的影响 |

## 环境设置

### 安装依赖

```bash
pip install jupyter numpy pandas matplotlib networkx
```

### 运行 Notebook

```bash
jupyter notebook
```

然后在浏览器中打开并选择相应的笔记本。

## 笔记本内容概览

### 01-introduction.ipynb
- 项目概述
- 社会科学模型解释
- 快速开始示例
- API 介绍

### 02-consensus-detection.ipynb
- 公识注入实验
- 弃权模式分析
- 检测准确率评估
- 结果可视化

### 03-influence-analysis.ipynb
- 少数派影响力阈值
- 压迫程度对投票行为的影响
- 群体动力学可视化
- 实验参数调优

## 使用建议

1. 从 `01-introduction.ipynb` 开始，了解基本概念
2. 运行简单的模拟实验
3. 修改参数观察不同结果
4. 阅读源代码了解实现细节
